المزامنة السحابية بين الأجهزة

لمزامنة البيانات بين جهازين فوراً، أنشئ مشروع Firebase مجاني (خطوات بالأسفل) وأدخل بياناته هنا. يجب استخدام نفس الإعدادات ونفس معرّف المتجر على كِلا الجهازين.
⚪ غير متصل — التطبيق يعمل بالتخزين المحلي فقط
📋

خطوات إنشاء مشروع Firebase (مرّة واحدة)

  1. افتح console.firebase.google.com وسجّل دخول بحساب Google
  2. اضغط "Add project" وأنشئ مشروعاً جديداً (أي اسم)
  3. من القائمة الجانبية: Build ← Firestore Database ← Create Database ← اختر "Start in test mode"
  4. من ⚙️ Project settings ← انزل لقسم "Your apps" ← اضغط أيقونة الويب () ← سجّل التطبيق
  5. انسخ القيم الظاهرة (apiKey, authDomain...) والصقها بالحقول أعلاه، ثم اضغط "حفظ والاتصال"
  6. كرر نفس الخطوات (أو انسخ نفس البيانات) على الجهاز الثاني، مع كتابة نفس معرّف المتجر بالضبط
⚠️ ملاحظة أمان: "test mode" يسمح لأي شخص يملك بيانات مشروعك بالقراءة والكتابة لمدة ٣٠ يوماً. لمتجر صغير بين شخصين هذا مقبول مبدئياً، لكن يُفضّل لاحقاً ضبط "Firestore Rules" لتقييد الوصول بكلمة مرور أو مصادقة.
